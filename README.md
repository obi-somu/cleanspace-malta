# CleanSpace Malta

A simple static one-page profile website for CleanSpace Malta, a cleaning service concept based in Malta.

## Website sections

- Home
- Services
- Price guide
- Service areas
- Booking message template
- Contact buttons

## Service focus

CleanSpace Malta is positioned for rooms, studios, shared apartments, small flats, move-in cleaning, and central Malta cleaning support.

## Before publishing

Update these placeholders in `index.html`:

- WhatsApp link: `https://wa.me/35600000000`
- Email address: `hello@cleanspacemalta.com`
- Prices, if required
- Service areas, if required

## Deployment

This project is ready for GitHub Pages, Netlify, Vercel, or any static hosting provider.
